💾 Simpan Data ke Google Sheets
Data peserta, semua skor mentah, SS, dan komposit akan dikirim ke spreadsheet Anda.
URL disimpan otomatis di browser ini — tidak perlu diisi ulang setiap kali.
📋 Cara setup Google Apps Script (klik untuk lihat)
1. Buka Google Sheets Anda → klik menu Extensions → Apps Script
2. ⚠️ Jika Anda SUDAH PERNAH setup sebelumnya (ada fungsi readSetting, getRecord, setupMindsetApp, dll), JANGAN hapus semua kode! Cukup cari fungsi doPost(e) yang sudah ada, hapus fungsi itu saja, lalu tempel penggantinya di bawah ini (fungsi lain seperti doGet, readSetting, setupMindsetApp, jsonResponse dibiarkan seperti apa adanya):
function doPost(e) {
  try {
    const payload   = JSON.parse(e.postData.contents);
    const tes       = (payload.tes || '').toUpperCase();
    const sheetName = tes === 'NST' ? SHEET_NST : SHEET_TKD;

    const ss    = SpreadsheetApp.getActiveSpreadsheet();
    let   sheet = ss.getSheetByName(sheetName);
    if (!sheet) sheet = ss.insertSheet(sheetName);

    if (sheet.getLastRow() === 0 && payload.headers) {
      sheet.appendRow(payload.headers);
      const hr = sheet.getRange(1, 1, 1, payload.headers.length);
      hr.setFontWeight('bold');
      hr.setBackground('#D6E4F0');
      sheet.setFrozenRows(1);
      sheet.setColumnWidth(1, 160);
      sheet.setColumnWidth(3, 180);
    }

    // MODE UPDATE (fitur Edit Data) — cari baris dg timestamp = row_key, lalu timpa
    if (payload.mode === 'update' && payload.row_key && payload.values) {
      const data    = sheet.getDataRange().getValues();
      const headers = data[0] || payload.headers || [];
      const tsCol   = headers.indexOf('timestamp');
      let rowIndex  = -1;
      if (tsCol > -1) {
        for (let i = 1; i < data.length; i++) {
          if (String(data[i][tsCol]) === String(payload.row_key)) { rowIndex = i + 1; break; }
        }
      }
      if (rowIndex > 0) {
        sheet.getRange(rowIndex, 1, 1, payload.values.length).setValues([payload.values]);
        return jsonResponse({ status: 'ok', sheet: sheetName, row: rowIndex, mode: 'updated' });
      }
      sheet.appendRow(payload.values);
      return jsonResponse({ status: 'ok', sheet: sheetName, row: sheet.getLastRow(), mode: 'appended' });
    }

    if (payload.values) sheet.appendRow(payload.values);
    return jsonResponse({ status: 'ok', sheet: sheetName, row: sheet.getLastRow() });

  } catch(err) {
    return jsonResponse({ status: 'error', message: err.message });
  }
}
📦 Setup dari nol (belum pernah ada Apps Script)? Minta file lengkap "Google Apps Script v0.4" ke pengembang — berisi doGet (baca setting/data), doPost (simpan/update), dan setupMindsetApp() (isi otomatis sheet "setting", "HASIL_TKD", "HASIL_NST").
3. Klik Deploy → Manage Deployments → klik ikon pensil (edit) pada deployment aktif → Version: New version → Deploy
4. (Kalau ini deployment pertama kali) Deploy → New Deployment → Type: Web App | Execute as: Me | Access: Anyone → copy URL → paste di kolom atas
5. Pertama kali: klik Authorize access dan login akun Google Anda
